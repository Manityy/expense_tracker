# Flousi

**Flousi** (*flous* — money in Tunisian Arabic) is a personal finance app built with Flutter. Track spending in **Tunisian Dinars (DT)**, set budgets, manage subscriptions, and get AI-powered advice tailored to your monthly income and goals.

---

## Features

### Home
- Monthly overview: salary, spent, remaining, and savings goal progress
- Spending forecast and personalized insights
- Per-category budget bars with over-budget alerts
- Recent expenses and quick access to the AI assistant

### Expenses
- Add, edit, and delete expenses with date picker
- Category chips, search, and grouping by day
- Swipe-to-delete and receipt photo (camera or gallery)
- Pull-to-refresh with live Firestore streams

### Analytics
- Month picker with summary cards (spent, remaining, % of income, transaction count)
- Category breakdown with pie chart and progress bars
- 6-month spending bar chart
- **Monthly Trend** — history, averages, and month-over-month changes

### Subscriptions
- Recurring expenses (rent, bills, streaming, etc.)
- Auto-post due subscriptions on app launch

### Profile
- Salary and savings goal setup
- Per-category budget limits
- Personalized spending challenges and badges
- Export expenses to CSV

### AI Assistant (Flousi AI)
- Chat powered by [Groq](https://groq.com/) with context from your salary, spending, and goals
- Conversation history saved in Firestore
- Advice framed in DT and your real monthly data

### Other
- Onboarding flow for new users
- Local notifications for budget alerts and challenges
- Firebase Auth (email/password, forgot password)
- Firestore security rules scoped per user

## Getting started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Dart ^3.12)
- A [Firebase](https://firebase.google.com/) project with Auth, Firestore, and Storage enabled
- A [Groq API key](https://console.groq.com/) (optional — AI chat requires it)

### 1. Clone and install

```bash
git clone https://github.com/<your-username>/expense_tracker_.git
cd expense_tracker_
flutter pub get
```

### 2. Firebase setup

This repo includes `lib/firebase_options.dart` and platform config files generated by FlutterFire. If you use your own Firebase project:

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

Enable **Email/Password** sign-in in the Firebase Console.

Deploy security rules before using the app in production:

```bash
firebase deploy --only firestore:rules,storage:rules
```

### 3. Environment variables

Create a `.env` file in the project root (gitignored):

```env
GROQ_API_KEY=your_groq_api_key_here
```

Alternatively, pass the key at build time:

```bash
flutter run --dart-define=GROQ_API_KEY=your_key_here
```

Without a valid Groq key, the rest of the app works; only the AI assistant will show a configuration message.

### 4. Run the app

```bash
flutter run
```

For Android release builds, core library desugaring is enabled in `android/app/build.gradle.kts` (required by local notifications).

## Firestore data model (overview)

| Collection | Purpose |
|------------|---------|
| `users/{uid}` | Profile: name, email, salary, savings goal, category budgets, onboarding flag |
| `users/{uid}/conversations/{id}` | AI chat threads |
| `users/{uid}/conversations/{id}/messages/{id}` | Chat messages |
| `users/{uid}/recurringExpenses/{id}` | Subscription / recurring expense definitions |
| `expenses/{id}` | Individual expense documents (`userId`, title, amount, category, date, optional receipt URL) |
